📌. Project Overview
This project is a Machine Learning-based system that predicts house prices using important features such as area, number of bedrooms, bathrooms, garage capacity, and house age.
The goal of this project is to demonstrate a complete ML pipeline including data preprocessing, feature engineering, model training, evaluation, and prediction.

📊 Dataset
Dataset used: House Prices Dataset (Ames Housing)
Source: Kaggle
The dataset contains multiple features related to residential homes such as size, location, and condition.

⚙️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

🧠 Machine Learning Models
The following models were implemented and compared:
Linear Regression
Random Forest Regressor
